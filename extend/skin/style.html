Style Bundle

The main visual aspects of a skin is provided by its style sheet (css). This style sheet has to be provided by a bundle fragment whose fragment host is com.vaadin. See Ripla Skins for more information.

The Ripla application defines a list of styles for the various visual components it uses. These styles can be used in the style sheet:

  • ripla-window
  • ripla-main
  • ripla-body
  • ripla-content
  • ripla-split
  • ripla-sidebar
  • ripla-menu
  • ripla-menubar
  • ripla-menu-item
  • ripla-menu-active
  • ripla-toolbar
  • ripla-toolbar-label
  • ripla-contextmenu
  • ripla-contextmenu-item
  • ripla-header
  • ripla-header-text
  • ripla-footer
  • ripla-footer-text
  • ripla-login-form
  • ripla-form
  • ripla-buttons
  • ripla-input
  • ripla-select
  • ripla-help
  • ripla-view
  • ripla-language-select
  • ripla-lookup
  • ripla-lookup-close
  • ripla-label
  • ripla-value
  • ripla-value-emphasized

See the Demo skin's stylesheet for an example.
